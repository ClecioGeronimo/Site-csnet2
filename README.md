# CSNET PRO LINK

Modern website for an Internet Service Provider built with React, TypeScript, and Tailwind CSS.

## Features

- 🚀 Fast and responsive design
- 💻 Modern tech stack (React, TypeScript, Tailwind CSS)
- 🛍️ Built-in e-commerce for networking products
- 📱 Mobile-first approach
- 🔍 SEO optimized
- 🌐 Internet speed test functionality
- 📦 Service plans showcase
- 📍 Coverage area information
- 💬 WhatsApp integration for customer support

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router DOM
- Lucide React for icons

## Getting Started

### Prerequisites

- Node.js 16.x or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/csnet-pro-link.git
cd csnet-pro-link
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
